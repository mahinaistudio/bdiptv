# BD IPTV

BD IPTV allows users to stream Bangladeshi TV channels online using an M3U playlist with any IPTV player.

Many free M3U links available on the internet stop working because they are not maintained regularly. Development of many such repositories has been discontinued, so no support or updates should be expected. However, since this source is open, anyone can fork the repository and manage it independently.

The playlist can be used with various IPTV players on Windows, Android, iOS, Apple TV, and Android TV.

## User Guide

### Step 1: Install an IPTV Player

Choose an IPTV player based on your device.

### Step 2: Get the M3U Playlist

* Copy the **M3U URL**, or
* Download the **M3U file** directly

### Step 3: Add the Playlist to the Player
**‚ñ∂Windows**

* MyIPTV Player

1. Open MyIPTV Player
2. Go to **Settings**
3. Select **Add new playlist and EPG source**
4. Paste the M3U URL or load the M3U file
5. Save and refresh the playlist

* VLC Media Player

1. Open VLC
2. Click **Media ‚Üí Open Network Stream**
3. Paste the M3U URL
4. Click **Play**

**‚ñ∂Android**

* IPTV Player

1. Open the IPTV app
2. Choose **Add Playlist**
3. Select **Add via URL** or **File**
4. Paste the M3U link or upload the file
5. Load the channels

**‚ñ∂iOS / iPad / Apple TV**

* IPTV Player-Play M3U Playlist

1. Open the IPTV Player app
2. Tap **Add Playlist**
3. Choose **M3U URL**
4. Paste the playlist link
5. Save and start watching
